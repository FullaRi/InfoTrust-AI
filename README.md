# InfoTrust AI

Projet académique – Atelier pratique en cybersécurité II  
Détection automatique et explicable de la désinformation par intelligence artificielle.

## Objectif
Rendre visible et compréhensible le processus de détection des fake news à l’aide de modèles d’IA et d’agents intelligents.

## Structure du projet
- `frontend/` : Interface web utilisateur
- `backend/` : Logique de traitement IA
- `n8n/` : Orchestration des agents IA (workflows)
- `data/` : Jeux de données (locaux uniquement)
- `logs/` : Journaux d’événements horodatés
- `docs/` : Documentation et schémas

## État du projet
- Jalon 1 : Analyse, état de l’art, cadrage 
- Jalon 2 : Développement du prototype
